# 淺藍特色互動按鈕核對紀錄

主頁「開啟懸浮視窗」已依照參考按鈕的雙光暈位移概念，改為淺藍至天藍的漸變按鈕；互動設計保留現有玻璃擬態、文字對比與主頁功能。

| 核對項目 | 結果 |
| --- | --- |
| 桌面版 | 1280 × 720；按鈕在卡片中置中，文字、音量圖示與右側方向標記完整可見；右側天藍柔焦光暈形成明顯但不干擾文字的層次。 |
| 手機版 | 375 × 812；按鈕、文字和兩側圖示均位於可視範圍內，沒有裁切或水平溢出。 |
| 互動狀態 | 懸停時雙光暈位移、圖示微幅旋轉與放大、文字底線及方向標記位移皆以 CSS transition 處理；按壓縮放為 0.98。 |
| 可及性 | 保留原有 `aria-label`，加入 `data-state` 表示 ready／open 狀態、明顯 `:focus-visible` 焦點環，並在 `prefers-reduced-motion: reduce` 停用非必要過場。 |
| 自動化檢查 | 7 個測試檔、39 項測試均通過；TypeScript 一次性檢查通過。 |

> 參考元件的雙柔焦圓形與懸停位移理念來自 [Uiverse 的 Fuzzy Rabbit 3](https://uiverse.io/Javierrocadev/fuzzy-rabbit-3)，實作已改用本專案的淺藍漸變、類別命名與可及性狀態。

## 公開版本核對

以版本參數重新載入公開網址後，主頁已顯示「開啟懸浮視窗」的淺藍漸變按鈕、左側音量圖示與右側方向標記，確認公開部署已載入最新前端資源。
